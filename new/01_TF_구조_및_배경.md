# 01. TF 구조 및 배경

---

## TF 단계별 히스토리

### Phase 0 — TF 결성 (2025년 9월 말)
- 기획자 4명으로 구성
- 가설 없이 독립 소재 제작 방식으로 시작
- UA 크리에이티브와 성과 지표 간 연결 체계 부재

### Phase 1 — 초기 집행 (2025년 9~11월)
- 소재를 집행하되, 어떤 요인이 IPM에 영향을 미치는지 파악하지 못한 상태
- 마케팅팀과의 역할 경계가 불명확한 시기

### Phase 2 — 개인별 가설 시대 (2025년 12월 ~ 2026년 3월)
- 마케팅팀 요청으로 가설 개념 도입
- 각 기획자가 개별적으로 상이한 가설을 운용
- 대조군 불일치, 변수 미통제, 시즌성/에버그린 가설 혼재 등 신뢰도 문제 발생

### Phase 3 — 구조적 모순 공식화 (2026년 3월)
- 본부장께 "현재 TF 구조가 최선인가?" 질문 제기
- 마케팅팀이 플레이북 방식 제안
- 마케팅 전문가 TF 합류 확정

---

## 구조적 한계 진단

### IPM 결정 변수와 TF 통제 범위 불일치

IPM을 결정하는 4가지 변수:

| 변수 | 통제 주체 |
|------|-----------|
| 크리에이티브 품질 | **TF (통제 가능)** |
| 타겟팅 | 마케팅팀 |
| 캠페인 구조 | 마케팅팀 |
| 예산 | 마케팅팀 |

> **문제**: TF가 IPM을 단독 목표 지표로 설정받았으나, 실제로는 4개 변수 중 1개(크리에이티브)만 통제 가능한 구조. 이는 개인 간 해결 가능한 문제가 아닌 **구조적 문제**로, 공식적 해결이 필요.

### 가설 데이터 신뢰도 문제 (Phase 2 원인)
- 대조군이 테스트마다 달라 비교 기준 불일치
- 시즌성 소재와 에버그린 소재가 같은 DB에 혼재
- 여러 변수를 동시에 변경하여 인과관계 파악 불가

---

## TF 목표 재정의 (2026년 3월 회의 결정)

- **기존**: UA 크리에이티브 IPM 극대화
- **재정의**: UA + 리텐션 포함 전체 마케팅 성과 극대화

---

## 핵심 원칙

1. **구조적 문제는 구조적으로 해결** — 개인 간 비공식 합의가 아닌 본부장 레벨 공식화 필요
2. **재검증 우선** — 새 가설 생성보다 기존 가설 재검증으로 데이터 신뢰도 축적
3. **단일 고정 대조군** — 모든 테스트의 기준점이 되는 소재를 하나로 고정
